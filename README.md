🎓 Almahyyr AI – Interactive Arabic Learning Platform for MTs Students
aplikasi teman belajar yang menyenangkan

Almahyyr AI is an AI-powered interactive Arabic language learning platform developed using Python, Streamlit, Google Gemini AI, and Google Text-to-Speech (gTTS). The application is designed to support Grade 8 Madrasah Tsanawiyah (MTs) students in learning Arabic through engaging, multimodal, and conversational learning experiences.

The platform integrates Artificial Intelligence to provide personalized learning assistance, interactive conversations, listening activities, speaking assessments, and Arabic vocabulary practice. By combining modern educational technology with communicative language learning principles, Almahyyr AI creates a more engaging and student-centered learning environment.

📖 Learning Modules

The application provides four primary learning modes:

Mufradat (Vocabulary Learning)
Maharah Istima' (Listening Skills)
Maharah Kalam (Speaking Skills)
Interactive AI Conversation

Each module is designed to enhance students' communicative competence in Arabic by integrating vocabulary acquisition, listening comprehension, pronunciation practice, and conversational interaction.
